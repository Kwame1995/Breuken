Hoofdstuk 3

Delen

Delen door een breuk is vermenigvuldigen met het omgekeerde:

3/4 : 2/5 =
3/4 x 5/2

Boven de streep: 3 x 5 = 15
Onder de streep: 4 x 2 = 8
invullen: 15/8

voorbeeld opgaves:

1) 6/2 : 6/5 = 30/12 2) 5/2 : 7/2 = 10/14 3) 2/4 : 6/3 = 6/24 4) 4/9 : 2/5 = 20/18 5) 12/7 : 13/5 = 60/81
